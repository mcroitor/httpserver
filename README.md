# httpserver
Simple HTTP server.

## Build
Just make:

```bash
$ make
```
The folder `build` will be created with `server` application.

## Start

The project contains default site folder `html`, you can customize files in it. For running do:

```bash
$ make start
```

And access http://localhost from your browser.
